# Ejemplo de Documento Markdown

Este es un archivo de prueba para convertir a PDF o DOCX.

## Características principales

- Conversión rápida
- Procesamiento en memoria
- Sin base de datos
- Deploy en Vercel

## Uso

1. Selecciona o arrastra un archivo `.md`
2. Elige el formato de salida (PDF o DOCX)
3. Descarga el archivo convertido

### Código de ejemplo

```javascript
const mensaje = "Hola Mundo"
console.log(mensaje)
```

## Formatos soportados

**Negrita**, *itálica*, y ***ambos***.
